An EPITECH project constiting of recreating a fully fonctional Shell program made in C with 4 of my friends.

42SHELL

This project lasted over 5 months and was the most challenging project i've ever done.

Despite having less than a year of experience in C and in team work (because school team projects are not really their priority), my team and I managed to create an entire terminal, that is almost fully fonctional. The only features missing are the arrow navigation and history up-arrow.

I'm really proud of this project because a friend of mine and myself were the members who brought new features to implement in the main program. It was the most technically challenging code writing i have ever performed and these features made me progress really quickly and catch up to the really good developper in my promo.

An RPG like indie-game programmed in C with CSFML library as my final first-year project at EPITECH

project:kaben

This indie game with a story based on our lovely school EPITECH Strasbourg was made in only 1 month with the C graphical library CSFML. Fastidious project to manage but we are very proud of the game we proceeded to create.

I was tasked to create the main architecture of the game, implementing the technical foundation for the game to even have a story. I've entirely created a custom map built like our school and made the fight versus our directeur? Mohammed Kaben.
